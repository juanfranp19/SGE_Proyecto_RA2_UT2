# 05 — Instalación de Odoo

1. Ejecutamos el instalador (`.exe`) como **Administrador**. Nos aparecerá este mensaje para permitir que haga cambios en el dispositivo. Le decimos que **Sí**.

![Ejecutar instalador](../assets/img/05-instalacion_odoo/paso01_ejecutar-instalador.png "Ejecutar instalador")

2. Seleccionamos el **idioma**.

![Seleccionar idioma](../assets/img/05-instalacion_odoo/paso02_seleccionar-idioma.png "Seleccionar idioma")

3. Nos da la **bienvenida**.

![Bienvenido a Odoo](../assets/img/05-instalacion_odoo/paso03_welcome-to-odoo.png "Bienvenido a Odoo")

4. Leemos los **términos** y decimos que estamos de acuerdo "**I agree**".

![Términos](../assets/img/05-instalacion_odoo/paso04_terminos.png "Aceptar términos")

5. **PASO IMPORTANTE**. Nos aseguramos de que instalamos **Odoo Server** y **PostgreSQL Server**: tienen que estar los dos opciones marcadas con un tic verde.

![Instalar Odoo Server y PostgreSQL Server](../assets/img/05-instalacion_odoo/paso05_odoo-server-y-postgres-server.png "Instalar Odoo Server y PostgreSQL Server")

6. **Configuramos** la **conexión** de PostgreSQL. 

> **Recomendación**: *Username* y *Password* sean **iguales**

![Configuración de PostgreSQL](../assets/img/05-instalacion_odoo/paso06_postgres-configuracion.png "Configuración de PostgreSQL")

7. Finalmente **instalamos Odoo**. No tenemos por qué cambiar la ruta.

![Instalar](../assets/img/05-instalacion_odoo/paso07_instalar.png "Instalar")

8. Esperamos a que se **instale por completo**. Acto seguido continuamos con el procedimiento.

![Instalando](../assets/img/05-instalacion_odoo/paso08_instalando.png "Instalando")

![Instalado](../assets/img/05-instalacion_odoo/paso08_instalado.png "Instalado")

9. Le damos a **Finish** y nos ejecutará Odoo en la web con un **puerto local**, por defecto **8069**. En el punto [08-Creación de base de datos de prueba](./08-creacion_bd_prueba.md) crearemos la primera **base de datos**.

![Ejecutar Odoo](../assets/img/05-instalacion_odoo/paso09_ejecutar-odoo.png "Ejecutar Odoo")
