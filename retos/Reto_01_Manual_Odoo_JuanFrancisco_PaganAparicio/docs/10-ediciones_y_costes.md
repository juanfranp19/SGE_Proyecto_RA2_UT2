## Odoo Community

- Versión gratuita.
- Incluye solo una aplicación.
- Usuarios ilimitados.

## Odoo Estándar

- Versión de pago.
- Incluye todas las aplicaciones.
- Incluye Odoo Online.
- Precio por usuario.
- Administración y actualizaciones automáticas gestionadas por Odoo.

## Odoo Personalizado

Incluye lo mismo que Odoo Estándar, más:

- Servicio SaaS.
- Odoo Studio.
- Soporte para múltiples empresas.
- Integración mediante API externa.

## Contratos

- **Mensual**:
	Pagas mes a mes y cancelas cuando quieras.

- **Anual**:
	Pagas mes a mes con menor precio que el contrato mensual, pero con un compromiso de 12 meses.

## Prueba gratuita

Para las versiones de pago, puedes contar con una versión de prueba gratuita con algunas condiciones:

- Duración de **15 días**.
- Límite inicial de 10 aplicaciones, cuando hayas instalado Odoo puede instalar más.

Lo demás es exactamente igual como pone en el plan elegido.
